About the Interstellar Armada application

This application is a learning project in the early development phase. The current version is only intended for testing the initial, already implemented features of what one day might become a functional game. Even the core features are yet under development. The application might contain numerous bugs.

About the used technologies

This project is written in JavaScript, without the usage of third party libraries. WebGL is used for 3D graphics, with shaders written in GLSL.
HTML5 and CSS3 are used for generating and styling the pages seen in the game.
The configuration files use the XML format, and the 3D model geometries are stored in the egm (EgomModel) file format, which is also well formed XML generated by a modeling library developed by the same author.
