Class PhysicalObject

The basic entity for all physical simulations. Can have physical properties and interact with other objects.
Defined in: physics.js.

Method Detail
addOrRenewForce(forceID, strength, direction, duration)
Checks the forces for one with the given ID, if it exists, renews its properties, if it does not, adds a new force with the given parameters. It will renew all forces with the given ID, if more than one exists.
Parameters:
{String} forceID
The ID of the force to look for
{number} strength
The strength of the force.
{number[]} direction
The vector describing the direction of the force.
{number} duration
The force will either created with, or renewed to last for this duration.
