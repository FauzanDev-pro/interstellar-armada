Class Detail
Scene(left, top, width, height, clearColorOnRender, colorMask, clearColor, clearDepthOnRender, lodContext)
Creates a new scene graph object.
Parameters:
{number} left
The X coordinate of the top left corner of the viewport on the screen.
{number} top
The Y coordinate of the top left corner of the viewport on the screen.
{number} width
The width of the viewport in pixels.
{number} height
The height of the viewport in pixels.
{boolean} clearColorOnRender
Whether to clear the color buffer every time at the beginning of rendering the scene.
{boolean[]} colorMask
Which components shall be cleared if the color buffer is to be cleared.
{number[]} clearColor
What color to use when clearing the buffer (RGBA components).
{boolean} clearDepthOnRender
Whether to clear the depth buffer every time at the beginning of rendering the scene.
{LODContext} lodContext
The LOD threshold and configuration to be used for rendering object with the appropriate level of detail.
Method Detail
assignUniforms(gl, shader)
Assigns all uniforms in the given shader program that the scene has a value function for, using the appropriate webGL calls. The matching is done based on the names of the uniforms.
Parameters:
{object} gl
The webGL context to use
{Shader} shader
The shader program in which to assign the uniforms.
render(resourceCenter)
Renders the whole scene applying the general configuration and then rendering all visual objects in the graph.
Parameters:
{ResourceCenter} resourceCenter
The resource center that holds the shaders, textures, models of the scene.
