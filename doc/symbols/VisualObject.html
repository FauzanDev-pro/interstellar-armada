Class VisualObject

The parent class of all objects to be rendered in the scene graph such as full viewport quads, shaded meshes, billboards or particle systems. Serves as a node, contains references to its parent and subnodes as well.
Defined in: graphics.js.

Parameters:
{Shader} shader
The shader that should be active while rendering this object
{number} smallestParentSizeWhenDrawn
If the rendering parent's apparent size is smaller than this value, render will not take place.
{boolean} renderedWithDepthMask
Tells whether this object should be rendered when the depth mask is on (= it contains non-transparent triangles)
{boolean} renderedWithoutDepthMask
Tells whether this object should be rendered when the depth mask is off (= it contains transparent triangles)
