{Float32Array} inverseRotationMatrix(m)
Calculates and returns the inverse of the top left 3x3 block of a 4x4 matrix, but complemented to 4x4 matrix (using identity matrix values for the rest part). Can be used to calculate the inverse of a rotation (or scaling) described in the 3x3 part of a 4x4 matrix without letting the optional translation interfere with it.
Defined in: matrices.js.

{Float32Array} m
The input 4x4 matrix.
