Class Resource

Ancestor class for all classes representing resources that need to be prepared (e.g. loaded from external source) before they can be used. As the loading happends asynchronously in many cases, this class provides a safe way to interact with the objects at any time, queuing the actions if the resource is not ready yet to use.
Defined in: resource.js.

{Boolean} executeWhenReady(functionToExecute, functionToExecuteIfNotReady)
Executes the first given function if the resourse is ready, otherwise queues it to be executed when it gets ready. Optionally takes a second function to be executed right now in case the resource is not ready yet to execute the first one (such as notifying the user).
Parameters:
{Function} functionToExecute
The function to execute when the resource is ready (now or later).
{Function} functionToExecuteIfNotReady Optional
The function to be executed if the resource is not ready yet.
Returns:
{Boolean} True if the first function got executed, false if it got queued.
