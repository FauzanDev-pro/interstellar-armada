Class ResourceCenter

This class holds and manages all the various resources and their configuration that are needed for rendering: textures, shaders, models, cameras, scenes.
Defined in: graphics.js.

{EgomModel} addModel(model, name)
Checks if the model passed as parameter is already in the resource center, and if not, then adds it. Also uses name checking, and doesn't add the model if another one if already present with the same name.
Parameters:
{EgomModel} model
The model resource we are looking for in the resource center.
{string} name
The name of the model resource we are looking for.
Returns:
{EgomModel} The found or added model object in the resource center.
