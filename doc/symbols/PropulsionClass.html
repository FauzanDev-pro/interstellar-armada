Class PropulsionClass

Each spacecraft can be equipped with a propulsion system. This class represents one of the classes to which such a system can belong, describing the properties of such a propulsion system.
Defined in: logic.js.

PropulsionClass(name, shaderName, textureFileName, color, thrust, angularThrust)
Defines a propulsion class.
Parameters:
{String} name
When describing the equipped propulsion system, it's class has to be referred to by this name.
{String} shaderName
The shader that will be used for rendering the particles shown when thrusters of the ship fire.
{String} textureFileName
The file to be used for the texture of the thruster particles.
{Number[3]} color
The color that can be used to modulate the color of thruster particles, if defined so by the shader. [red,green,blue]
{Number} thrust
The strength of the force applied to the ship when the thrusters are fired in one direction, measured in newtons.
{Number} angularThrust
The strength of the torque applied to the ship when the thrusters are used to turn it.
Returns:
{PropulsionClass}
