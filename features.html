Implemented features

Graphics

  • cubemapped background
  • loading 3D models from XML
  • texture mapping with arbitrary color modulation
  • simple shading with directional lights
  • vertex luminosity data
  • models with multiple Level Of Detail (LOD)
  • support for line and triangle based drawing
  • camera facing and 1 axis fixed, turning billboards
  • multiple cameras with adjustable FOV
  • hierarchic mesh structure
  • frustum culling of meshes to optimize performance
  • alpha blending using vertex color data and transparent png textures (*only for billboards yet)
  • loading shader configuration from an XML file, easy shader setup in the program source

Physics

  • simplified newtonian physics set up accordingly for space environment (no drag)

Game logic

  • loading definition of in-game object classes from XML (including automated load and setup for referenced model, texture and shader files)
  • loading full level configuration from XML (*excluding lighting information for now)
  • possibility to specify equipped weapons on individual spaceships

Gameplay

Planned features

Major planned features of the game

  • highly scalable performance and control to automatically adapt to a wide range of devices from phone to PC

  • diverse weapons for space dogfight

  • huge world created with the power of sophisticated and smart random generators

  • diverse quests (missions) created by a complex random generator, put together from many schemes and elements
