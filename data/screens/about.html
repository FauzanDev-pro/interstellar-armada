Used software

This game is created using freely available and/or open source software and open standards. For creating the in-game assets, Inkscape, Blender, Gimp and specialized tools developed (in Netbeans and Lazarus) by Krisztián Nagy are used, and the game itself is being developed using Netbeans, Google Chrome and Firefox, all of these running on Ubuntu.
